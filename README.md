# Zsámboki Biokert — Weekly Report Tool

Internal commercial reporting tool for Zsámboki Biokert.

## Usage

Open the live tool at: `https://<your-username>.github.io/<repo-name>/`

Upload your weekly order export, map columns, and generate a PDF report covering:
- Orders by pickup point (with 2025 rolling average comparison)
- Social media analytics (Facebook & Instagram)
- Top performing Facebook posts
- Product breakdown

All data is processed locally in your browser. Nothing is sent to any server.

## Updating the tool

1. Download the updated `index.html` from Claude
2. Replace the file in this repo (drag and drop on GitHub or use `git`)
3. GitHub Pages will redeploy automatically within ~30 seconds
